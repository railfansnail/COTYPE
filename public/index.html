1COTYPE에 오신 걸 환영합니다!

COTYPE(코타입)은 실제 코드를 직접 쳐보면서 타자 실력과 코딩 감각을
동시에 키울 수 있는 개발자 전용 타자 연습 플랫폼입니다.

⌨
실제 코드 타이핑
HTML·CSS·JS·Python·C·C++·C#·Java 실제 코드를 직접 타이핑하며 연습합니다.
📊
WPM & 타수 측정
실시간으로 WPM(단어/분)과 타수(글자/분), 정확도를 측정합니다.
🔤
단어 연습 모드
언어별 핵심 키워드만 골라서 빠르게 손에 익히는 단어 연습 모드가 있습니다.
🏆
글로벌 랭킹
연습 완료 후 닉네임으로 랭킹에 등록하고 다른 사람과 기록을 겨뤄보세요.
💡 이 튜토리얼은 언제든 상단 ? 도움말 버튼으로 다시 볼 수 있습니다.

2화면 구성 안내

화면은 크게 3개 영역으로 나뉩니다.

① 상단 바 (Topbar)
현재 파일명, 실시간 WPM·타수/분·정확도·진행률을 표시합니다. HTML 연습 시 ◧ 미리보기 버튼으로 오른쪽에 실시간 HTML 렌더링을 띄울 수 있습니다.
② 왼쪽 사이드바
언어 선택(HTML·CSS·JS·Python·C·C++·C#·Java)과 코드 / 단어 모드 전환, 스니펫(예제 코드) 목록이 있습니다. ＋ 스니펫 추가로 나만의 코드를 등록할 수도 있습니다.
③ 중앙 에디터
타이핑할 코드가 표시됩니다. 아무 곳이나 클릭하거나 키보드를 누르면 자동으로 포커스가 맞춰집니다. 문법 강조 색상이 적용되어 가독성을 높여줍니다.
💡 오른쪽 상단 🏆 랭킹 버튼으로 전체·언어별 랭킹 보드를 확인할 수 있습니다.

3타이핑 방법

화면의 코드를 그대로 따라 타이핑하면 됩니다. 글자 색상으로 상태를 확인하세요.

const name = "홍길동";
bad 오타 (빨간 반전) cur 현재 커서 위치 dim 아직 미입력 (회색)

오타를 입력해도 백스페이스로 수정할 수 있습니다. 실수해도 괜찮아요!

Tab들여쓰기 (다음 공백 자동 삽입)
Enter줄바꿈 입력
Backspace한 글자 수정
Esc처음부터 다시 시작
Alt+R재시작
Alt+N다음 스니펫으로 이동
💡 한글 입력도 정상 지원됩니다. 조합 중인 글자는 확정 후 비교하므로 자연스럽게 입력하세요.
🔒 연습창에서는 복사·붙여넣기가 차단됩니다. 부정행위 방지를 위한 조치입니다.

4모드 & 언어 선택

사이드바 상단에서 코드 또는 단어 모드를 선택할 수 있습니다.

📄
코드 모드
실제 코드 스니펫 전체를 처음부터 끝까지 타이핑합니다. 들여쓰기·특수문자 포함 실전 감각을 키울 수 있습니다.
🔤
단어 모드

5랭킹 등록 & 학습 팁

코드 타이핑을 완료하면 결과창이 뜨고, 🏆 랭킹 등록 버튼으로 기록을 남길 수 있습니다.

등록 절차
① 타이핑 완료 → ② 결과 확인 (WPM·타수·정확도) → ③ "랭킹 등록" 클릭 → ④ 닉네임 입력 → ⑤ 내 순위 확인
⚠ 랭킹 등록 제한
복붙 수준의 비정상적인 속도(WPM 300 초과 또는 5초 내 50자 이상)가 감지되면 랭킹 등록이 자동으로 차단됩니다.

📈 실력 향상 팁

✅ 정확도 우선: 처음엔 느려도 오타 없이 치는 연습부터 시작하세요. 속도는 자연스럽게 따라옵니다.
✅ 단어 모드 활용: 새로운 언어를 시작할 때 단어 모드로 키워드를 먼저 손에 익히면 코드 모드가 훨씬 쉬워집니다.
✅ 같은 스니펫 반복: 어려운 스니펫은 Esc로 초기화 후 반복 연습하세요. 기억에 오래 남습니다.
